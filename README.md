# Homebrew CircleCI

The official CircleCI Homebrew tap for macOS tools and packages.

## Installation

Add the tap to Homebrew:

```shell
$ brew tap circleci-public/homebrew-circleci
```

## Packages

### Casks

| Cask | Description |
|------|-------------|
| `circleci-runner` | CircleCI Self-Hosted Machine Runner for macOS |

See the [Casks README](Casks/README.md) for installation and configuration details.

## Contributing

Pull requests for CircleCI packages are gladly reviewed. When opening a pull request please ensure the following points are included in the pull request description:

- The nature of the changes made
- The reasoning for the changes
- Testing done for changes
