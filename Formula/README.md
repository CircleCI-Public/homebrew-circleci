## chunk

The Chunk CLI is a CircleCI tool for splitting test files across parallel runners to optimize test execution time.

See the [chunk-cli repository](https://github.com/CircleCI-Public/chunk-cli) for more information.

### Installation

```shell
$ brew install chunk
```
